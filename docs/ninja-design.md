# นินจา (Ninja) — อาชีพพิเศษที่ 4 · Design Doc v1

> สถานะ: **SHIPPED (2026-07-08, 5 waves)** — เจ้าของเคาะรูปทรง + sim ใช้สิทธิ์ปรับตัวเลข (ร่างเดิมอ่อนเกิน ไม่ใช่แรงเกิน) — **ตัวเลขจริงที่ ship ดูที่ `docs/balance-ninja.md`** (hpMult 1.35, massacre 40 mana, eternal 72/14/×9.0, ratio 4DEX:1VIT:1INT); ตัวเลขในเอกสารนี้คือร่างแรกเชิงประวัติศาสตร์
> เจ้าของเคาะแล้ว (2026-07-08): อาชีพพิเศษปลดล็อกเมื่อทั้ง 3 อาชีพหลักถึง tier 3 · ช่องตัวละคร 3→4 · สาย tier: นินจา → จอมนินจา → ราชันเงา · ปล่อยรวมกับ patch เสี่ยจ๋อง

## 1. ตัวตน (identity)

| | ดาบ | ธนู | เวท | **นินจา** |
|---|---|---|---|---|
| ระยะ | ประชิด | ไกล | ไกล | **ประชิดสั้นสุด** |
| จังหวะ | ปานกลาง อึด | ยิงเร็ว | ช้า AoE หนัก | **ตีถี่สุดในเกม** |
| การเคลื่อนที่ | เดิน | เดิน+kite | เดิน | **dash/กระพริบเข้าเป้า** |
| สเตตัสหลัก | STR | DEX | INT | **DEX** (สายประชิด) |

- **อาวุธ**: มีดสั้นคู่ (dagger) — โจมตีปกติ "ตี 2 ครั้งต่อจังหวะ" ครั้งละ ~55% ของ ATK (DPS รวมสูงกว่าดาบเล็กน้อย แลกกับระยะสั้นสุด + ตัวบางกว่า)
- **กลไกใหม่ของ engine**: `dash` — ตัวละครย้ายตำแหน่งไปข้างเป้าหมายทันที (deterministic, ไม่มี RNG, ระยะจำกัดต่อครั้ง, ยิง event `heroDashed {fromX, toX}` ให้ render วาดเส้นเงา) ใช้ร่วมโดยสกิล 1/3/4
- **แฟนตาซีหลัก** (คำเจ้าของ): "กระโดดฟันเคลียมอนไปมาอย่างรวดเร็ว"

## 2. สเตตัสฐาน + auto-allocate

- ฐาน: `{ str 5, dex 8, int 3, vit 4 }` (รวม 20 — ระดับเดียวกับธนู 20 / ดาบ 21 / เวท 19)
- DEX ของนินจา = ATK ประชิด + อัตราโจมตี (reuse กติกา DEX เดิมฝั่ง ranged แต่ผูกกับ dagger)
- Auto-allocate ratio ร่าง: **3 DEX : 1 VIT** — sim เป็นผู้เคาะจริง (บทเรียน auto-allocate v2: sim เคยคว่ำร่างมาแล้ว)

## 3. สกิล 4 ท่า (ตัวเลขร่างแรก — sim จูน)

| # | ชื่อ | ปลดล็อก | รูปแบบ | ตัวเลขร่าง |
|---|---|---|---|---|
| 1 | **เงาพริบ** | เริ่มต้น | dash ทะลุเป้าใกล้สุด + ฟัน 1 ที | ×1.8 DMG · มานา 20 · CD 4s |
| 2 | **คมเงาคู่** | Lv6 | รัว 5 ทีใส่เป้าเดียว + สาดข้างเคียง r80 ครึ่งแรง | 5×0.6 (+splash 50%) · มานา 35 · CD 8s |
| 3 | **เงาสังหาร** (ultimate tier 2) | เปลี่ยนเป็นจอมนินจา | **โซ่ dash ไล่ฟันทีละตัวสูงสุด 8 ตัวทั่วสนามใน ~1.5s** — ลายเซ็นอาชีพ | ×1.4/เป้า · มานา 90 · CD 20s |
| 4 | **พันเงานิรันดร์** (skill 4 tier 3) | ราชันเงา Lv40 | จอสลัว + เงาแยกร่างฟัน**ทุกตัวในสนาม** + time-freeze สั้น + ตัวจริงโผล่กลางวง | ×2.2/เป้า · มานา 170 · CD 45s |

- Spectacle ท่า 3/4 ใช้เครื่อง skyDarken/curtain/time-freeze เดิม (ไม่มี ProjectileKind ใหม่ = ไม่ต้องเพิ่ม render map... ยกเว้น fx เส้นเงา dash เป็น fx module ใหม่)
- มานา: อยู่กลางสเปกตรัม (แพงกว่าดาบ ถูกกว่าเวท) — นินจาควร "กินยามานาพอรู้สึก" ตามหลัก mana = pacing governor

## 4. สายอาชีพ

นินจา (tier 1) → **จอมนินจา** (tier 2, เควสเดิมรูปแบบเดียวกับอาชีพอื่น) → **ราชันเงา** (tier 3 @Lv40, เควสรูปแบบ tier-3 เดิม) — ได้สกิล 4 + ช่อง auto-cast ที่ 4 ตามแบบแผน

## 5. การปลดล็อก + ช่องตัวละคร

- เงื่อนไข: **บัญชีมีตัวละคร 3 อาชีพหลัก (ดาบ/ธนู/เวท) ที่ถึง tier 3 ครบทุกอาชีพ** — ตรวจฝั่ง server (เพิ่มคอลัมน์ cache `Character.tier` ประทับตอนเซฟ แบบเดียวกับ level/power — additive, db push)
- การ์ดสร้างตัวละครใบที่ 4 โชว์ตลอดแบบ locked พร้อมความคืบหน้า ("ปลดล็อก: อาชีพ tier 3 แล้ว 2/3")
- `MAX_LIVE_CHARACTERS 3 → 4` + copy ทุกจุดที่พูดว่า "สูงสุด 3 ตัว"
- เผื่ออนาคต: เงื่อนไขปลดล็อกอยู่หลัง flag เดียว ถอดง่าย (เจ้าของ: "ภายหลังค่อยปลดเงื่อนไขนี้")

## 6. Gear

- มีดสั้น **t1-t10** (~12 templates ตามจำนวนของสายอาวุธอื่น) `classReq: "ninja"` · เกราะใช้ระบบเดิม (ไม่ผูกอาชีพ)
- ⚠️ **ข้อควรระวังเชิงวิศวกรรม**: การเพิ่มของเข้า drop table เปลี่ยนลำดับ loot roll ที่ deterministic — ต้อง rebaseline canonical sim อย่างตั้งใจ (เหมือนตอน M7.9 เพิ่ม t7-10) หรือกรอง drop dagger เฉพาะตัวละครนินจา — ตัดสินใจใน wave gear โดยยึด "ผู้เล่นเดิมกระทบน้อยสุด" และประกาศใน commit ชัด ๆ

## 7. Render

- Rig ใหม่: เงาบาง คล่อง ผ้าพันคอ/ผ้าคาดหัว มีดคู่สองมือ · ท่าโจมตีสลับซ้าย-ขวา · dash = เส้นเงา + afterimage สั้น · ออร่า tier 2/3 ตามแบบแผนอาชีพอื่น
- ภาษา fx สายเงา: เส้นฟันสีเงิน-ม่วงเข้ม (jewel-tone ตาม art direction) บนพื้นหลัง desaturated

## 8. Balance gates (มาตรฐานเดิมทุกข้อ)

- เคลียร์ s1-30 โซโล่ไม่มีกำแพงถาวร · ชนะเควสบอส tier 2/3 · อัตรากินยาอยู่ในย่านที่ตั้ง · อาชีพเดิม 3 อาชีพ **byte-identical** (ยกเว้น rebaseline จาก drop table ถ้าเลือกทางนั้น — ประกาศชัด)
- DPS target: สูงกว่าดาบ ~10-15% ชดเชยความบาง/ระยะสั้น — sim ตัดสิน

## 9. แผนงาน (5 waves หลัง doc ผ่าน)

1. **Engine** (ชิ้นใหญ่สุด): HeroClass +"ninja", dash primitive, สกิล 4, เควส tier 2/3, SAVE v18 migration
2. **Gear/drops**: dagger templates + นโยบาย drop table (ข้อ 6)
3. **Render**: rig + dash fx + ออร่า
4. **UI + Server**: การ์ดปลดล็อก + ช่อง 4 + `Character.tier` cache + i18n/codex/FTUE/HOF filter
5. **Sim**: gates ทั้งชุด + auto-allocate ratio + จูนตัวเลขสกิล
