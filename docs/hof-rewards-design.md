# รางวัล Hall of Fame ประจำงวด — Design Doc v1 (เจ้าของเคาะ 2026-07-08)

> สถานะ: **รูปทรงเคาะแล้วทั้งหมด** ("ok ครับถูกใจ ลุยได้เลย") — คิวงาน: implement หลังนินจา 5 waves จบ
> หลักการแม่: รางวัล HOF **ไม่มีวันเป็นพลังถาวร** (กติกา GDD: power = level + stats + class + gear เท่านั้น) — ของแพงสุดคือ "หน้าตาและชื่อเสียง"

## 1. โครงงวด

- **งวดรายเดือน** ตัดยอด **เที่ยงคืนไทย (Asia/Bangkok) สิ้นเดือน** — ใช้แกน serverDay เดิมของ dailies
- แชมป์ครองฉายา/ออร่า/ป้ายเมืองตลอดงวดถัดไป แล้ววัดใหม่จากศูนย์ (บอร์ดสะสมอย่าง level/gold วัดค่ารวม ณ วันตัด — ไม่รีเซ็ตค่าจริงของผู้เล่น)
- ตัวละครที่ anti-cheat ซ่อน (suspect) หลุดบอร์ดอยู่แล้ว = ไม่มีสิทธิ์รับรางวัลโดยอัตโนมัติ ไม่ต้องมีกติกาเพิ่ม

## 2. หมวดที่มีรางวัล (4 หมวด) + ตารางฉายา

| หมวด | 🥇 อันดับ 1 | 🥈 อันดับ 2 | 🥉 อันดับ 3 | รางวัลพิเศษอันดับ 1 |
|---|---|---|---|---|
| ระดับสูงสุด | จ้าวยุทธภพ | ยอดยุทธ์ | จอมยุทธ์ | ออร่าทอง + ป้ายเมือง + แกร่งสุ่ม 50:50 ×1 |
| พลังสูงสุด | ผู้แข็งแกร่งแห่งปฐพี | กำปั้นเหล็ก | ผู้ฝึกตน | ออร่าทอง + ป้ายเมือง + แกร่งสุ่ม 50:50 ×1 |
| ทองสะสม | เสี่ยใหญ่ | เสี่ยรอง | เสี่ยฝึกหัด | ออร่าทอง + ป้ายเมือง + แกร่งสุ่ม 50:50 ×1 |
| ออนไลน์นานสุด | หัวหน้ายาม | ยามอาวุโส | ยามตัวน้อย | — ฉายาอย่างเดียว (เจ้าของ: "คงไม่ให้อะไรเยอะ") |

- **ล้มบอสไวสุด: ตัดออกจากรอบรางวัล v1** (เจ้าของ: รายละเอียดเยอะ บอสหลายโซน คิดรางวัลยาก) — บอร์ดยังแสดงใน HOF ตามเดิม; ฟื้นรางวัลได้ภายหลังเป็น backlog
- ฉายาโชว์ที่: ป้ายชื่อในฉาก (nameplate) / ปาร์ตี้ / หน้า HOF / ประกาศ — ติดหลายหมวดพร้อมกันได้ แต่**เลือกโชว์ 1 ฉายา** (ตั้งที่ Settings)

## 3. องค์ประกอบรางวัล

1. **ฉายา** (อันดับ 1-3 ทุกหมวด) — ครอง 1 งวด หมดงวดวัดใหม่
2. **ออร่าแชมป์สีทอง** (อันดับ 1 ยกเว้นออนไลน์) — render-only, ภาษาภาพต้องต่างจากออร่า +8/+9/+10 prestige และออร่า tier ชัดเจน
3. **ป้ายเกียรติยศกลางเมืองหลัก** — สลักชื่อแชมป์ปัจจุบันทุกหมวด (decor แนวเดียวกับลามะ; ศิลป์รอชุด pixel art ของเจ้าของถ้ามี — ไม่มีก็ป้ายวาดเอง)
4. **แกร่งสุ่ม 50:50** (แกร่งอาวุธ/แกร่งชุด ×1, อันดับ 1 ของ 3 หมวดแรก) — mint ผ่าน pipeline เดียวกับรางวัลเสี่ยจ๋อง (ItemInstance kind "fortifier", บอทห้ามขาย, ขายตลาดกลางได้ในอนาคต)
5. **ตราถาวรสะสม** (แชมป์อันดับ 1 ทุกหมวดรวมออนไลน์) — ประทับลงโปรไฟล์ถาวร เช่น 🏅 "จ้าวยุทธภพ · ก.ค. 2026"; เสียแชมป์แล้วตราไม่หาย; สะสมข้ามเดือน/หมวด; เห็นได้เมื่อกดดูโปรไฟล์จาก HOF ("แชมป์ N สมัย") — ไม่ใช่ไอเทม ขายไม่ได้ ไม่กินกระเป๋า
   - จิตวิทยาคู่: ฉายา = แย่งกันเดือนนี้ · ตรา = ดึงแชมป์เก่ากลับมาต่อสถิติ

## 4. รูปแบบ implement (ร่างสำหรับ wave จริง)

- **ปิดงวดแบบ lazy-finalize** (โปรเจกต์ไม่มี cron): request แรกที่ข้ามเส้นตัดยอดเป็นตัว trigger สรุปงวด ใน tx เดียว + ledger unique กัน double-finalize — แบบแผนเดียวกับ DailyClaim/WorldBossClaim
- ตาราง additive ใหม่ (db push รอบเดียวกับของค้างอื่น): `HofSeason` (เดือน, สถานะ finalized) + `HofAward` (seasonId, board, rank, characterId, title, claimedAt) — award เป็นทั้งบันทึกตรา (ถาวร) และสิทธิ์รับแกร่ง (claim idempotent)
- ฉายาปัจจุบัน = derive จาก award ของงวดล่าสุดที่ finalized (ไม่ persist ซ้ำ) · ฉายาที่เลือกโชว์เก็บใน uiConfig ต่อ character
- Nameplate: ต่อผ่าน seam `setHeroDisplayNames` เดิม (ขยาย payload ชื่อ+ฉายา) — engine ไม่รู้จักฉายา (render/ui-only, ไม่มี SAVE bump)
- อันดับใช้ข้อมูลบอร์ด HOF ที่ anti-cheat re-derive แล้วเท่านั้น
- Edge cases: หมวดว่าง/คนไม่ครบ 3 = แจกเท่าที่มี · ตัวละครถูกลบก่อนรับ = award ตกไป (ตราไม่ mint ให้ตัวที่ลบ — สอดคล้อง deleted-char purge เดิม) · เสมอกันใช้ tie-break เดิมของบอร์ด

## 5. Backlog เฟสหน้า (เจ้าของฝากไว้)

- **ไอเทมรางวัลสุดพิเศษผูกตลาดกลาง** — ของสะสมมีใบเดียวในโลกต่อหมวดต่อเดือน (แนวถ้วย/มงกุฎ) ให้ design ใหม่ตอนตลาดกลางเป็นรูปเป็นร่าง (ข้อเสนอถ้วยรอบแรก "ยังไม่โดนใจ" — ต้องพิเศษกว่านี้)
- ฟื้นรางวัลหมวดล้มบอสไวสุด
