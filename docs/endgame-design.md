# Endgame Loop v1 — Design Doc (ร่างรอเจ้าของเคาะ)

> กรอบที่เคาะแล้ว (2026-07-08): s30 จบใน ~2 ชม. · แกน = ระบบวนซ้ำ (ค) + แมพโหดแบบประหยัดสำหรับสาย AFK (ก-lite) · เกณฑ์ความโหด: **ตี +8 ขึ้นไปยืนสบาย (รวมบอทกดยา)** ต่ำกว่านั้นลำบาก · tier 4 พัก

## 1. แมพโหด "ดินแดนอสูร" (ก-lite — เนื้อหาใหม่ราคาถูก)

- **Reuse โซน s1-30 เดิมทั้งหมด** เป็นเวอร์ชันโหด (s31-60 ภายใน): มอน/บอสตัวเดิม สเตตัสคูณขึ้น, render แค่ย้อมโทนสี biome (เลือด/มืด) + ป้ายชื่อ "อสูร" — ไม่ต้องวาดโลกใหม่
- **ปลดล็อก**: เคลียร์ s30 แล้วเท่านั้น (ประตูใหม่ในเมือง หรือ warp picker แท็บใหม่)
- **เกณฑ์จูน (sim gate ใหม่ HARD=1)**: ตัวละคร L60-70 ตี +8 ฟาร์มยืนได้ deaths ต่ำ · +6-7 เหนื่อยชัด · +5 ลงไป = กำแพง — ตรงคำเคาะเป๊ะ
- **รางวัล = throughput ไม่ใช่ power ใหม่**: xp ต่อเนื่อง (คว้าเส้นทาง L60→**levelCap 90** ที่มีอยู่แล้วแต่ไม่มีที่ฟาร์ม), drop t9-t10 อัตราดีขึ้น, หินเสริมกอง, gold สูง — **ไม่มี tier อาวุธใหม่** (เพดาน t10+10 คงเดิม = ไม่ทำลาย balance เดิม)
- สาย AFK: เปิดบอททิ้งไว้ได้เหมือนแมพปกติทุกอย่าง (บอท/เควส/ตี้ ใช้ระบบเดิมหมด)

## 2. แกนวนซ้ำ (ค — ผูกของที่มีให้เป็นวงจรรายวัน/รายสัปดาห์/รายเดือน)

| จังหวะ | ระบบ | สถานะ |
|---|---|---|
| รายชั่วโมง | เสี่ยจ๋อง (+ อนาคต: world boss หมุนเวียนในแมพโหด ตัวใหม่ ๆ) | มีแล้ว / ต่อยอด |
| รายวัน | เควสรายวัน 3 ใบ + ล่าแกร่ง/หิน | มีแล้ว |
| รายสัปดาห์-เดือน | ฤดูกาล HOF (ฉายา/ออร่า/ป้ายเมือง) | มีแล้ว |
| ไม่มีที่สิ้นสุด | สายตี +10 ครบทุกชิ้น + ไต่ L90 + สะสมตราแชมป์ | มีแล้ว + แมพโหดเติมเชื้อ |
| อนาคต (เฟสถัดไป) | ตลาดกลาง (ค้าขาย = endgame ถาวร) + "หอทดสอบ" boss-rush สำหรับตี้ | ยังไม่ทำ |

## 3. เศรษฐกิจ

gold/หินไหลแรงขึ้นจากแมพโหด + ยาราคาเดียว → **เงินเฟ้อโดยรู้ตัว** (เจ้าของ: "เดี๋ยวหากิจมาละลายเป็นช่วง ๆ") — จุดระบายที่วางไว้: event ละลายทอง, ตลาดกลาง (ค่าธรรมเนียม), ทบทวนก่อนเปิดตลาด

## 4. แผนงาน (4 waves หลังเคาะ)

1. **Engine**: hard-zone table (คูณสเตตัส/รางวัลต่อโซน), ปลดล็อกหลัง s30, SAVE v19 (persist ความคืบหน้าแมพโหด), sim HARD=1 + เกณฑ์ +8
2. **UI/Server**: ประตู/แท็บ warp, goal card, codex, i18n, patch notes (server: แค่ zone stamp เดิมรองรับ)
3. **Render**: ย้อมโทน biome + ป้ายอสูร (ถูกที่สุด — reuse ทุกอย่าง)
4. **Sim canonical**: gate +8 ทุกอาชีพ, s1-30 เดิม byte-identical, ตาราง TTK/รายได้ต่อชั่วโมง

ประมาณเวลา: ~4-6 ชม. งาน agent | คำถามเปิดให้เจ้าของ: ชื่อโหมด ("ดินแดนอสูร"?), จำนวนขั้น (โหดขั้นเดียว หรือ อสูร I/II/III), world boss ตัวใหม่ในแมพโหดเอาเลยไหมหรือเฟสหน้า

## v1.1 — เจ้าของปรับสเปค (2026-07-08)

- **แมพโหดยาว 10 โซน/แมพ** (จากเดิมโครง 5) — เริ่ม 1 แมพ (s31-40 ภายใน)
- **ดรอป: ไม่มีของใหม่ใน v1** — เจ้าของขอคิดเอง (โครงรองรับเติมทีหลังแบบ class-gated ได้โดยไม่ rebaseline)
- **เกณฑ์ความโหดใหม่ (แทนของเดิม)**: **+8 ยืนเกือบไม่รอด · +9/+10 ยืนสบาย · ต่ำกว่า +8 = กำแพง** (รวมบอทกดยา) — sim gate วัด deaths ต่อระดับตีบวก
- ขั้นเดียว ไม่มี I/II/III ใน v1 · ไม่มี world boss ใหม่ใน v1
- **มุมเสนอเพิ่ม (รอเคาะ): "ลึกแค่ไหน = ตีบวกถึงไหน"** — ไล่ความโหดใน 10 โซน: z1-3 เข้าได้ที่ +8 (ปริ่มน้ำ), z4-7 ต้อง +9, z8-10 ต้อง +10 → เส้นทางตีบวกกลายเป็นแผนที่จริง ยิ่งบวกยิ่งเดินลึก รางวัล (xp/gold/หิน) ไต่ตามความลึก

## v1.2 — Legendary crafting "ตำราตำนาน" (เจ้าของเคาะโครง 2026-07-08)

- **เมนูคราฟแยกเดี่ยว** "⚒️ ตำราตำนาน" ในเมนูหลัก: checklist 4 ช่องแบบเรียลไทม์ + ปุ่มคราฟ
- **สูตร (ผูกกับแมพอสูรทั้งหมด)**: แก่นอสูร ×N (Elite ดรอป, นับได้ไม่มีดวง, stream แยกแบบหิน) + ศิลาโซน ×10 (หีบ first-clear ต่อโซน ครั้งเดียวตลอดชีพ — บังคับไต่สุดแมพ) + ตราอสูร ×3-4 (บอส z10 รางวัลจำกัดสัปดาห์ละ 1) + **อาวุธ t10 อาชีพตัวเองเป็นวัตถุดิบตั้งต้น** (ตีความ "เริ่มที่ t10" ของเจ้าของ = สังเวยเข้าเตา, sink t10 ส่วนเกิน) + ค่าตีทอง/หิน (sink เงินเฟ้อ)
- **พลัง**: ~t10×1.4 + สายปลุกพลัง +0..+5 ของตัวเอง (ใช้สูตร refinedStat, ไม่มีแตก) เพดาน +9-19% เหนือ t10+10 · ไม่ใช้ตีบวกปกติ · sim gates: t10+10 ยังเคลียร์อสูรครบ, ไม่มี legendary = byte-identical
- **ผูกตัวละครตอนคราฟ** (เจ้าของรับข้อเสนอ) · ประกาศเซิร์ฟเวอร์คนแรกต่ออาชีพ · render เส้นแสง+particle เฉพาะอาชีพ (ดาบเถ้าเพลิง/ธนูดาวตก/เวทรูน/นินจาเงา)
- **เฟส**: อสูร v1 = แมพ 10 โซน (ไล่ลึกตามตีบวก +8/+9/+10) + Elite+แก่น + หีบ first-clear + โซนฮอตรายวัน (#1 ถ้าเอา) → patch คราฟตามหลัง (เมนู+สูตร+SAVE bump+render)
- ค้างเคาะสุดท้าย: คำสั่งเริ่มสร้าง v1
